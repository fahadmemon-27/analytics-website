About This Platform

This website is built for Web Analytics lab experiments. It tracks user behavior like page visits, clicks, time spent, and form submissions.

What this website collects

  • ✅ Page visits (Home/About/Contact/Analytics)
  • ✅ Time spent on each page
  • ✅ Image clicks and button clicks
  • ✅ Navigation usage
  • ✅ Form submission analytics

Why this is useful

The recorded interaction logs are displayed in a structured format, which makes scraping simple and helps in performing analytics experiments.
